# Zee's Tasting Circle

Open the page through a web server, not by double-clicking `index.html`.

For local testing:

```powershell
python -m http.server 8000
```

Then open:

```text
http://localhost:8000/
```

The waitlist form uses FormSubmit and sends submissions to `zikrakakhana@gmail.com`. The first submission may require activating the FormSubmit email confirmation.
